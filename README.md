# 生活アプリ

日常の詰まりを取るための個人用Webアプリです。GitHub Pagesにアップロードして、iPhoneのホーム画面に追加できます。

## 入っている機能

- 今日：そろそろ、今日の提案、5分/15分/30分/1時間の行動候補
- 買う：買い物リスト、ステータス、値段、底値
- 整える：美容医療、ヘア、脱毛、肌管理などの時系列ログ
- 減らす：デジタルデトックスの方法と効果記録
- メモ：やりたいことを一括メモ、行動候補生成、ChatGPT用プロンプトコピー

## 使い方

1. このフォルダ内の `index.html`, `style.css`, `app.js`, `manifest.webmanifest` をGitHubリポジトリにアップロードします。
2. GitHub Pagesを有効化します。
3. 公開URLをiPhoneのSafariで開き、「ホーム画面に追加」します。

## 注意

- データはブラウザのlocalStorageに保存されます。
- 端末やブラウザを変えるとデータは引き継がれません。
- 現時点の行動候補はAPIなしのルールベースです。ChatGPT用プロンプトをコピーして、より賢く整理できます。
