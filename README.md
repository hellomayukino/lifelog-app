# 今これ

生活ログを材料に、今この瞬間にできる一番小さい行動を出すためのWebアプリです。

## 使い方

1. GitHubで新しいリポジトリを作る
2. このフォルダ内のファイルを、ZIPではなく中身のままアップロードする
3. Settings → Pages → Branch: main / Folder: root にする
4. 表示されたURLを開く
5. iPhoneやPCブラウザで使う

## 入っている機能

- 今これ：時間・状態に合わせた小さい行動候補
- 材料：やりたいこと・気になることのブレインダンプ
- 買う：買い物メモ
- 整える：美容・体調・メンテ記録
- 記録：できた行動の履歴

## 注意

この版はAPIなしで動く、ルールベースの簡易版です。ChatGPTに貼る用のプロンプト生成は入っています。
